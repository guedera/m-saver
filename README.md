# m-saver

Plataforma web para acompanhamento de finanças pessoais mensais. Pensada para ser simples e acessível tanto no computador quanto no celular.

## O que é

O m-saver permite registrar entradas e saídas de dinheiro ao longo do mês — salário, mesada, compras, contas, etc. — e exibe um dashboard com um resumo visual do que foi gasto, com o que, e quanto sobrou.

## Funcionalidades previstas

- Registro de gastos com categoria e descrição
- Registro de entradas (salário, mesada, renda extra...)
- Dashboard mensal com resumo de entradas, saídas e saldo
- Gráficos de distribuição de gastos por categoria
- Navegação por meses anteriores
- Interface responsiva e amigável para celular
